MockServer Class

Defined in: sdk/utility/mock_server.js:1
Module: MockServer
Parent Module: AP

Mock server instances are used by the test framework to intercept XHR requests and simulate the functionality of an API backend, entirely offline. A mock server handles all requests matching collections within the application within which it is instantiated.

To enable a mock server in an application, simply enable the global useMockServer flag:

handleDeleteRequest

(
  • xhr
  • collectionClass
)

Provided by the utility module.

Defined in sdk/utility/mock_server.js:357

Handles an intercepted XHR DELETE request for the given collection class. The passed request body must be valid JSON and must contain an ID attribute. It is parsed and used to remove an existing model instance from the datastore.

If the requested instance exists, an empty XHR 204 response is returned.

If the requested instance does not exist, a mock XHR 404 response is sent.

Parameters:

  • xhr Object

    mock XHR request object generated by sinon

  • collectionClass AP.collection.Collection

    the collection class to which this request applies

)

Provided by the utility module.

Defined in sdk/utility/mock_server.js:291

Handles an intercepted XHR POST request for the given collection class. The passed request body must be valid JSON. It is parsed and used to instantiate a new model instance of the appropriate type using a randomly generated ID. The new model instance is added to the internal datastore and will be returned by future requests to the collection. The resulting model instances is JSON serialized and returned via a mock XHR 201 response.

Parameters:

  • xhr Object

    mock XHR request object generated by sinon

  • collectionClass AP.collection.Collection

    the collection class to which this request applies

Handles an intercepted XHR PUT request for the given collection class. The passed request body must be valid JSON and must contain an ID attribute. It is parsed and used to update an existing model instance of the appropriate type within the datastore.
