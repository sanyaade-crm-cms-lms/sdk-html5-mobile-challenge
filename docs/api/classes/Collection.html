Defined in sdk/collection/base_collection.js:85

Fetches objects from the collection instance's URL. Calls the underlaying Backbone Collection.fetch method. Note: data from the collection's optional {@link #extraParams} is passed through the URL of every request.

Parameters:

  • options Object

    optional request data

    • data Object

      optional request parameters are passed through request URL as-is

    • query Object

      optional query parameters are passed through request URL after being transformed by {@link #mapQuerParams}.

  • args... Object

    optional additional arguments passed-through to underlaying Backbone Collection.fetch method.

mapQueryParams

(
  • data
)
Object

Defined in sdk/collection/base_collection.js:64

Copies keys in object to keys of the format query[key_name] in a new object, where key_name is the original key. Returns a new object leaving the original intact. For example, an input object {foo: 'bar'} would result in an output object {query[foo]: 'bar'}.

Parameters:

  • data Object

    name/value pairs to map to query-format

Returns:

Object:

a new object with mapped keys
